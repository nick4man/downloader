# iOS: отправка ссылок в downloader через Shortcut

Safari/iOS не поддерживает Web Share Target (в отличие от Android), поэтому
вместо PWA используем **ярлык Shortcuts** — он появится в меню «Поделиться».

Готовый `.shortcut`-файл раздать нельзя (подписанный формат), но сборка — 1 минута.

## Сборка ярлыка
1. Открой приложение **Команды (Shortcuts)** → «+» (новый ярлык).
2. Внизу настройки ярлыка (ⓘ) → включи **«Показывать в меню «Поделиться»»**,
   тип входа — **URL** (и **Текст**).
3. Добавь действия по порядку:
   - **«Закодировать URL»** (URL Encode) над **«Вход ярлыка»** (Shortcut Input).
   - **«Текст»**:
     ```
     https://ТВОЙ-ДОМЕН/share?token=ТВОЙ-ТОКЕН&url=[Закодированный URL]
     ```
     (вставь результат предыдущего действия в `url=`).
   - **«Загрузить содержимое URL»** (Get Contents of URL), метод **GET**, на этот текст.
4. Назови ярлык, например, **«В downloader»**.

## Использование
В любом приложении (Safari, YouTube, …): **«Поделиться» → «В downloader»** —
ссылка уходит в очередь, демон начинает качать.

## Аутентификация
- **Токен**: подставлен в `?token=` (как выше). Самый простой путь для iOS.
- **Cloudflare Access**: если домен закрыт логином, ярлыку нужен сервисный токен
  Access (headers `CF-Access-Client-Id`/`CF-Access-Client-Secret`) — добавь их в
  «Загрузить содержимое URL» → «Заголовки». Для личного использования проще токен.

## Без своего домена
Нужен HTTPS-адрес демона (Cloudflare Tunnel из `DEPLOY.md`). По локальному
`http://…` из сотовой сети ярлык не достучится.
