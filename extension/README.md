# Send to downloader — браузерное расширение

Правый клик по ссылке / видео / странице → **«Отправить в downloader»** —
URL улетает в демон (`POST /jobs`), и тот сразу ставит его в очередь.

## Установка (Chrome / Edge / любой Chromium)
1. Открой `chrome://extensions`, включи **Режим разработчика**.
2. **Загрузить распакованное** → выбери эту папку `extension/`.
3. (Опционально) В настройках расширения укажи адрес демона, если он не
   `http://127.0.0.1:8765` (например `http://192.168.1.50:8765`).

## Firefox
Зайди на `about:debugging` → «Этот Firefox» → «Загрузить временное дополнение»
→ выбери `manifest.json`. (Для постоянной установки нужна подпись.)

## Удалённый демон
Расширение шлёт запрос на `base/jobs`. Для не-localhost адреса добавь его в
`host_permissions` в `manifest.json`, например `"http://192.168.1.50:8765/*"`,
и укажи тот же адрес в настройках расширения.

## Требование на стороне демона
Демон должен разрешать CORS для origin расширения — по умолчанию
`cors_origins = ["*"]` (или env `DOWNLOADER_CORS`). Это уже включено.

Обратная связь — бейдж на значке расширения: зелёная **✓** (принято) /
красная **✗** (ошибка).
